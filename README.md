# MERN-chating-App
MERN chating app is a Full Stack Chatting App. Uses Socket.io for real time communication and stores user details in encrypted format in Mongo DB Database.

## Tech Used

* React js
* Chakra UI
* Node JS
* Express JS
* Moongo DB
